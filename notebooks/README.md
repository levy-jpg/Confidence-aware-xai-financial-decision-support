This folder contains the original development pipelines behind the Confidence-Aware XAI Streamlit prototype.

## Notebooks

- `MSc_Project_Applicant_Profiles.ipynb`  
  Documents how curated applicant profiles were selected and exported for the participant-facing user-study prototype.

The notebooks save reusable app artefacts into the project-level `models/` and `data/` folders.
